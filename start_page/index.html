INFORMACIÓN

¿Qué es invisual Map?

InvisualMap es una innovadora aplicación web desarrollada para acompañar a las personas invidentes que asisten a la escuela especial Sonva, ubicada en el sector Sonva, Cuenca. Esta aplicación tiene como objetivo principal brindar una manera segura y confiable de navegar para los estudiantes de esta institución, quienes enfrentan el desafío diario de la discapacidad visual. Al estar enfocada en ofrecer asistencia y apoyo a las personas invidentes, InvisualMap se convierte en una herramienta indispensable para la comunidad de la escuela especial Sonva, proporcionando información detallada sobre rutas, puntos de interés y obstáculos potenciales.

Razón de creación de InvisualMap

InvisualMap fue creado para abordar las dificultades de movilidad que enfrentan las personas invidentes en Cuenca, especialmente en el sector de Sonva. Al observar la falta de acceso a dispositivos costosos, se desarrolló esta aplicación gratuita de última generación, con el propósito de proporcionar asistencia segura y confiable, mejorando así la calidad de vida y la independencia de la comunidad invidente en su día a día.

Funcionalidad

Esta cuenta con una interfaz intuitiva y accesible. Esta app de mapeo web facilita la movilidad y autonomía de los estudiantes invidentes, promoviendo la inclusión y mejorando significativamente su calidad de vida. Contiene:

  • Comandos de voz: Interacción fácil y accesible mediante comandos de voz.
  • Detección de obstáculos: Avisa sobre objetos cercanos para mayor seguridad.
  • Interacción intuitiva: Navegación clara y sencilla para una experiencia fluida.
  • Asistencia en tiempo real: Datos actualizados para recorridos informados.
  • Modo de exploración táctil: Experiencia personalizada para usuarios táctiles.
    • Con estas funciones, InvisualMap brinda seguridad y autonomía a las personas invidentes, facilitando su movilidad y promoviendo la inclusión en su día a día.

    Herramientas

    Para la implementación de InvisualMap, se emplearon diversas herramientas que fueron fundamentales para lograr la funcionalidad y accesibilidad deseada en InvisualMap, brindando una experiencia segura y confiable a la comunidad invidente en su rutina diaria, permitiendo ofrecer una experiencia de navegación completa y funcional para las personas invidentes:

  • Leaflet y OpenStreetMap: Creación del mapa interactivo con marcadores y obstáculos para una navegación detallada.
  • Strapi: Almacena datos de ubicación (latitud y longitud) y actúa como intermediario entre la base de datos y la página web.
  • API: Facilita la comunicación eficiente entre la aplicación web y la base de datos.
  • Vercel: Plataforma de alojamiento web para publicar InvisualMap y realizar pruebas de manera ágil.
  • HTML, CSS y JavaScript: Estructura, diseño visual y lógica interactiva de la aplicación.
  • OpenStreetMap: Plataforma que proporciona datos geográficos precisos para la navegación de los usuarios invidentes.
